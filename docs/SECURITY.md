# Security Policy

## Reporting a Vulnerability

If you discover a security vulnerability, please report it responsibly.

Please **do not open a public issue** for security vulnerabilities.

Instead, report it privately by:
- Opening a GitHub Security Advisory
- Contacting the maintainers through GitHub

The maintainers will review the report and respond as soon as possible.

## Disclosure Policy

After the vulnerability is fixed, the maintainers may disclose the issue publicly to help improve transparency and security awareness.
